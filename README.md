# 日本語 Vocab Sheet

Web interface for custom japanese and chinese study.

This tool is aimed at people who find flashcards to be tedious and prefer to study with a traditional word list. These lists are either decks that you may find on ankiweb or wordlists aggregated by internet users, but laid out in spreadsheet form.
Currently, the website provides the Core10k deck broken up into 10 parts of 1000 words each, as well as decks for N5, 4 and 3. There is also HSK1 through 6 for chinese language. You also get a backlog of words you can add from any deck for later review.


![alt text](https://i.imgur.com/VKjbqmC.png)


There are a number of features for self organizing your own study :

- **Clicking a sentence** shows furigana as long as the mouse is held
- **Alt + Click** adds an item to the backlog (or removes it)
- **Right click** opens a prompt for color coding : purple, blue or teal
- **Left slider** enable furigana on all kanji in the example sentences (column 5)
- **Right slider** hides kana and english translation
- **Sort button** sorts the rows based on color coding
- **Import / Export button** upload or download the color coding + backlog for archiving or use on another device


![alt text](https://i.imgur.com/QGvvNog.png)


The backlog and color code are saved in local storage, which means that they will be wiped if the cookies are erased.
